# vuex-elemantUI-boilerplate

## includes
 - [ ] Vuex
 - [x] element UI
 - [x] Webpack
 - [x] ESLint (standard)
 - [x] sass & pug

## Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).

``` bash
$ npm install -g vue-cli yarn
$ vue init uriuriuriu/vuex-element my-project
$ cd my-project
$ yarn install
$ yarn run dev
```
